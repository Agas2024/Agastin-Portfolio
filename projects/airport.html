Project Idea

Large organizations like airports manage hundreds of IT assets across multiple departments. Traditional manual inventory tracking leads to inefficiencies, data inconsistencies, and delayed reporting.

The idea of this project was to design a centralized digital system that tracks the movement of IT assets in real time, records in/out transactions accurately, and provides instant visibility into stock availability.

Technology Stack

  • Frontend: React.js, Vite
  • Backend: Node.js, Express.js
  • Database: MySQL
  • API Communication: Axios
  • Data Export: CSV / Excel

Key Features

  • Centralized in/out stock tracking for IT assets
  • Real-time inventory updates
  • Search and filter functionality
  • CSV export for reporting and audits
  • Responsive dashboard interface

Project Outcome

The system successfully replaced manual inventory tracking with a digital platform, improving accuracy, reducing redundancy, and enabling faster asset retrieval. The project provided a scalable foundation for future enhancements such as role-based access and barcode integration.
