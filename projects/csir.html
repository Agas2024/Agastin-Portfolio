Problem Statement

Manual inspection of wood defects in terahertz images is time-consuming, subjective, and error-prone. An automated, accurate, and real-time detection system was required to improve reliability and inspection speed.

Proposed Solution

  • Automated defect detection using YOLO-based deep learning models
  • Classification of defects such as knot, small knot, decay, and no-defect
  • Bounding-box localization for defect area estimation
  • High-accuracy detection on terahertz image datasets

Methodology

  • Dataset preparation and annotation using Roboflow
  • Training YOLO model on labeled terahertz images
  • Evaluation using precision, recall, and F1-score metrics
  • Performance analysis using training and validation loss curves

Key Results

  • Achieved F1-score greater than 0.97
  • Defect dimension estimation error below 4%
  • Consistent convergence across training epochs
  • Demonstrated feasibility of real-time industrial inspection

Publication

Artificial Intelligence-Driven Timber Wood Defect Characterization from Terahertz Images
